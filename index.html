    TC-LOGO

    This is a school project done in INSA Lyon. The main goal is to implement a simple LOGO compilator to SVG format.

    How it works

    The compiler uses flex and bison to parse and interprets the code. It represents it in a C struct to be able to easily navigate in the program. After representation, the compiler writes the SVG on the output.

    What can it do ?

    • Simple instructions like FORWARD, LEFT, RIGHT or REPEAT
    • Manage the pen by PU (pen up) and PD (pen down)

    It can be better

    • Implements macro for circle and square
    • Optimize SVG output
